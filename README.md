#TWG - Tecnowatt Gateway

Integrazione Home Assistant per TWG.
Board creata da Tecnowatt S.r.l.

## Installazione
- Tramite HACS (Custom Repository)
- Necessita di una instanza MQTT
- Categoria: Integration

## Configurazione
L'integrazione si configura da UI:
Impostazioni → Dispositivi e servizi → Aggiungi integrazione

## Entità supportate
- Sensor
- Binary Sensor
- Switch"# TWG-HACS-integration"
